The Math Behind the Magic

How does the computer always "read your mind"? It isn't magic—it is algebra!

1. The Algebra Formula

Any two-digit number can be written based on its tens digit (let's call it x) and its ones digit (let's call it y). The actual value of that number is:

Original Number = 10x + y

The game asks you to subtract both digits from your original number:

Result = (10x + y) - (x + y)

If we simplify this algebra equation, the ones digit (y) completely cancels itself out:

Result = 10x - x = 9x

Because the answer is always 9x, every single starting number yields a multiple of 9!

2. The Illusion Trap

No matter which number you start with, your final calculated result can only ever be one of these nine numbers:

9, 18, 27, 36, 45, 54, 63, 72, 81

If you look closely at the table, the game secretly assigns the exact same symbol to all nine of these numbers! When you click the magic square, it simply displays that pre-arranged symbol.
